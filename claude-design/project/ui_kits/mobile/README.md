# Mobile App UI Kit — 記録網
5〜10人の対面プレイ用スマホ画面のリファレンス。360×720 portrait 基準。
- JoinScreen — 参加 (QR + 6桁コード + 名簿)
- RoleScreen — 役職確認 (Private / PrivacyCover + HoldToReveal)
- NightScreen — 夜の対象選択 (Concealed 対応・全役職同一骨格)
- DayScreen — 昼の公開評議 → 裁定 (投票 + HoldToConfirm)
- ReconnectScreen — 切断からの復旧
- VictoryScreen — 終局 reveal + 役職名簿公開 + 再戦
index.html で全画面を JP/EN 切替つきで試遊できる。文字列は T(lang, jp, en) パターン。
